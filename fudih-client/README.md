# Vision AI Client

Next.js frontend for AI-powered chat with image analysis and generation.

## Setup

```bash
npm install
```

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=http://127.0.0.1:8000
```

## Run

```bash
npm run dev
```

## Features

- Real-time chat with SSE streaming
- Image upload and analysis
- Image generation support
- Dark mode support

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
